# Dulan Dhanush Portfolio Website

A personal portfolio website showcasing the projects, skills, and services of Dulan Dhanush, a software developer. The website highlights frontend and backend development skills, C# desktop applications, and UI/UX design experience.

## Overview

This responsive and interactive portfolio website is designed to present personal projects, technical skills, and professional experience. It features:

- Smooth scrolling and active navigation links  
- Responsive design compatible with desktops, tablets, and mobile devices  
- Interactive project cards with source code links  
- Typing animation for home section  
- Contact form handling with simulated submission feedback  
- Intersection animations for services and projects  
- Modern UI design with a focus on user experience

## Features

- **Home Section:** Highlights skills and a dynamic typing animation  
- **Services Section:** Showcases UI/UX, frontend, and backend development skills  
- **Projects Section:** Displays desktop and web projects with clickable source code buttons  
- **Contact Section:** Smooth scrolling contact button and form submission handling  
- **Responsive Design:** Works seamlessly across different screen sizes  
- **Animations:** Elements appear with smooth fade-in effects using Intersection Observer
